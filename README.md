# College Planner

A web application that helps students gather and organize all materials for their college applications in one place.

## Tech Stack

Frontend:
- React
- TypeScript
- Vite

Backend:
- Spring Boot (planned)

Database:
- PostgreSQL (planned)

Cloud:
- AWS
